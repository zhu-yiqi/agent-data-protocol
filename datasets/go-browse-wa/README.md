# Go-Browse-WA

This dataset obtained by running the Go-Browse data collection algorithm across 100 URLs from the WebArena websites.
It consits of ~10K successful and ~17K unsuccessful trajectories.
The dataset contains both screenshot and textual observations of webpages (e.g. accessibility tree, pruned html, etc.)
Please see the paper [Go-Browse: Training Web Agents with Structured Exploration](https://www.arxiv.org/abs/2506.03533) for more details.
