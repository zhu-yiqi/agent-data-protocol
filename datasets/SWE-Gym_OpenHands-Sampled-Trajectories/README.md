# SWE-Gym/OpenHands-Sampled-Trajectories

 [SWE-Gym](https://github.com/SWE-Gym/SWE-Gym) is an environment for training SWE agents, with 2.4K real tasks from 11 Python repos & a Lite split of 234 instances. SWE-Gym combines real-world Python tasks, repository context, executable environments, and test verification to train agents for solving software engineering problems.

[SWE-Gym/OpenHands-Sampled-Trajectories](https://huggingface.co/datasets/SWE-Gym/OpenHands-Sampled-Trajectories) is a dataset of 6,055 trajectories generated using [OpenHands](https://github.com/All-Hands-AI/OpenHands) and GPT-4o.
