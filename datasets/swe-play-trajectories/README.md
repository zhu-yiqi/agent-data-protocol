# SWE-Play-trajectories

The [StephenZhu/SWE-Play-trajectories](https://huggingface.co/datasets/StephenZhu/SWE-Play-trajectories) dataset contains 704 software engineering task trajectories generated by AI agents. The dataset captures interactions between an AI assistant (based on OpenHands) and an environment where the agent attempts to complete various programming tasks.

## Dataset Information

- **Original Dataset**: [StephenZhu/SWE-Play-trajectories](https://huggingface.co/datasets/StephenZhu/SWE-Play-trajectories)
- **Size**: 704 trajectories
- **Format**: Conversation format with system/user/assistant messages

## Data Structure

Each trajectory contains:
- **messages**: Array of conversation turns with `role` (system/user/assistant) and `content`
- **category**: Task classification (e.g., "general")
- **source**: Origin of the data (e.g., "cpu-architecture-simulator")

## Actions Converted

The dataset uses the following ADP actions:
- **ApiAction**: For structured tool calls (`str_replace_editor`, `think`, `finish`)
- **CodeAction**: For code execution (`execute_bash`, `execute_ipython_cell`)
- **MessageAction**: For plain text responses without tool calls

## Observations

- **TextObservation (source="user")**: Initial task descriptions
- **TextObservation (source="environment")**: Execution results from tool calls
