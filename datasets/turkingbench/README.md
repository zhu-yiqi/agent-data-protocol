
# TurkingBench

[TurkingBench](https://github.com/JHU-CLSP/turking-bench) is a benchmark of 158 real-world web-based tasks, originally designed for crowdsourcing, to evaluate the ability of AI agents to understand and interact with complex web pages.


# Steps to Process

1. **Install Dependencies**
     ```sh
     pip install -r requirements.txt
     ```

2. **Clone the turking-bench repo**
     ```sh
     cd datasets/turkingbench/
     git clone https://github.com/JHU-CLSP/turking-bench
     ```

3. **Run the Script**
     ```sh
     python extract_raw.py
     ```
